## Core Architecture
MyCitadel operates on a hardened **Near Zero-Knowledge (N-NZK)** framework.

* **Client-Side Sovereignty:** Data processing runs local mathematical operations directly inside your browser container before network transit occurs.
* **Blinded Transit:** Utilizing `Libsodium XChaCha20` for localized cryptographic operations to establish unbreakable, un-interceptable communication bridges.
* **Beacon Engine:** Anonymized telemetry tracks local interactions and masks sessions, ensuring audits and bug tracking without compromising individual privacy.
* **Default-Deny:** A strict architectural paradigm where no connection can be initiated or forced upon a user without explicit authorization.
